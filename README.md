# AI-BOX

AI 工具配置与技能仓库 — 一站式收集各种 AI 工具的接入指南、配置文件和使用技巧。

## 当前内容

### Skills（用于 Claude Code / Cowork）

| 技能 | 说明 |
|------|------|
| [claude-code-deepseek](skills/claude-code-deepseek/SKILL.md) | Claude Code CLI 接入 DeepSeek 后端：注册充值 → 环境变量 → 模型映射 → 验证排错，走完完整流程 |

## 使用方式

### Claude Code / Cowork 用户

1. 将此仓库克隆到本地
2. 将 `skills/` 目录下的文件夹复制到你的 skills 目录
3. 在 Cowork 中输入 `/claude-code-deepseek` 即可使用

### 直接参考

每个 skill 下的 `SKILL.md` 都是纯 Markdown，可以直接阅读照着操作。

## 为什么叫 AI-BOX

像一个工具箱，把各种 AI 工具的配置和经验装进去，随时取用。目前从 Claude Code + DeepSeek 起步，后续会陆续加入更多工具的指南。
